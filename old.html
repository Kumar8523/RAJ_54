Systematic Low-Cost Engineering Framework for Paper Aircraft

Transforming paper aircraft design from trial-and-error to systematic engineering with applications in micro-UAV development

Download Research Learn More

About the Research

This research develops and validates a systematic, low-cost engineering framework for optimizing paper aircraft through computer vision flight tracking and iterative design methodology.

Problem Statement

Traditional paper aircraft design relies on inefficient trial-and-error approaches that limit aerodynamic understanding and provide no systematic pathway to micro-UAV engineering skills.

Our Solution

By implementing the Lucidraft software for automated trajectory tracking, quantitative performance measurement, and enforced single-variable modification protocols.

Key Contributions

Transforms subjective flight assessment into reproducible engineering data with demonstrated transferability to micro-UAV development applications.

Framework Components

Our systematic approach combines engineering methodology with accessible technology

Systematic Design

Single-variable modification protocol for clear cause-effect relationships

Computer Vision Tracking

CSRT algorithm for automated trajectory analysis and performance metrics

Quantitative Metrics

Distance, airtime, speed, trajectory, and stability scoring (0-10 scale)

Data Management

Hierarchical storage with version control for all design iterations

Lucidraft Software

Our custom-developed software for implementing the systematic framework

Digital Engineering Notebook

Lucidraft serves as a digital engineering notebook with enhanced analytical capabilities, providing an organized folder structure with models/versions. Each version folder contains metadata, metrics, software-generated files, and other relevant information.

Key Features:

  • Automated flight tracking using CSRT algorithm
  • Trajectory visualization and pattern recognition
  • Performance metrics calculation and comparison
  • Version control and iteration management
  • Report generation and data export

Technical Requirements:

Standard computer capable of running Python applications with OpenCV support. No specialized hardware required beyond a basic webcam or smartphone for video recording.
